# flickr8k-multilabel-classification
Multi-label image classification on Flickr8k dataset using ResNet50, VGG16, and EfficientNet-B0 with TensorFlow/Keras.
A multi-label image classification system trained on the Flickr8k dataset using transfer learning. Implements and compares three pretrained CNN architectures — ResNet50, VGG16, and EfficientNet-B0 — fine-tuned with TensorFlow/Keras. Labels are extracted from image captions using NLP preprocessing. Achieves F1 scores above 80% with custom thresholding, evaluated using per-label classification reports and confusion matrices.
